## Descripción de la tarea

Esta tarea consiste en:

1. **Registrarse en GitHub**:
   - Acceder a [github.com](https://github.com).
   - Completar el formulario de registro (nombre de usuario, correo, contraseña).

2. **Crear el primer repositorio**:
   - Hacer clic en el botón **+** → **New repository**.
   - Ponerle un nombre
   - Marcar la opción **"Add a README file"** (opcional, pero lo haremos manualmente).
   - Hacer clic en **Create repository**.

3. **Añadir archivos**:
   - Subir o crear:
     - `README.md`: este archivo con la explicación.
     - `hola_mundo.py`: un script simple en Python que imprime "¡Hola, Mundo!".
